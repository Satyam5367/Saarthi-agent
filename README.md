## 🌟 Overview

Saarthi is an **AI-powered multi-agent banking assistant** designed to help banks proactively identify and reactivate dormant customer accounts. It combines intelligent reasoning, multilingual voice interactions, human handoff, and continuous learning to improve customer engagement while supporting financial inclusion.

---

# 🎯 The Problem

Millions of bank accounts become dormant because customers:

- Miss important banking notifications
- Face language barriers
- Relocate or change contact details
- Lack awareness of account status
- Cannot easily reach customer support

Banks incur higher operational costs while customers lose access to financial services.

---

# 🏗 Architecture

| Agent | Responsibility |
|-------|----------------|
| 🧠 Reasoning Agent | Understands customer context and plans outreach |
| 🎙 Voice Agent | Conducts multilingual conversations |
| 👨‍💼 Handoff Agent | Escalates complex cases to bank officials |
| 📈 Feedback Agent | Learns from interactions to improve future performance |

---

# 🔄 Workflow

1. Detect dormant accounts.
2. Analyze customer history.
3. Generate personalized outreach.
4. Conduct multilingual voice conversation.
5. Resolve or escalate issues.
6. Learn from outcomes.
